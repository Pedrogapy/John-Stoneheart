# John Stoneheart | Ficha Automatizada PHB 2024

Site estático em HTML, CSS e JavaScript para usar a ficha do John Stoneheart no navegador ou no GitHub Pages.

## O que foi ajustado pelo Livro do Jogador 2024

- Feiticeiro nível 3 com 3 Pontos de Feitiçaria, 4 truques de Feiticeiro, 6 magias de Feiticeiro preparadas, 4 espaços de 1º círculo e 2 espaços de 2º círculo.
- CD de magia calculada como 8 + bônus de proficiência + modificador de Carisma.
- Feitiçaria Inata adicionada: aumenta a CD em 1 e dá vantagem em ataques mágicos enquanto estiver ativa.
- Salvaguardas proficientes corrigidas para Constituição e Carisma.
- Feitiçaria Selvagem ajustada com Marés do Caos e Surto de Magia Selvagem no 20 do d20.
- Anão ajustado com Visão no Escuro 36 m, Resistência a Toxinas, Tenacidade Anã e Conhecimento de Pedras.
- Antecedente Sábio ajustado com Arcanismo, História, Suprimentos de Calígrafo e Iniciado em Magia (Mago).
- Magias antigas que não aparecem na lista de Feiticeiro 2024 foram mantidas como reserva/legado, marcadas para uso apenas se o mestre permitir.

## Como usar localmente

1. Extraia o `.zip`.
2. Abra o arquivo `index.html` no navegador.
3. A ficha salva automaticamente no armazenamento local do navegador.

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub.
2. Envie `index.html`, `style.css`, `script.js` e este `README.md` para a raiz do repositório.
3. Vá em **Settings > Pages**.
4. Em **Build and deployment**, selecione a branch `main` e a pasta `/root`.
5. Salve e aguarde o link do GitHub Pages.

## Observação

A tabela oficial completa de Surto de Magia Selvagem não foi copiada no site. O botão rola o d100 e manda consultar a tabela no livro.
